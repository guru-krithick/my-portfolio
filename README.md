## 🚀 Quick Start

1. **Clone the repository**
```bash
git clone https://github.com/guru-krithick/my-portfolio.git
cd my-portfolio
```

2. **Install dependencies**
```bash
npm install
# or
pnpm install
```

3. **Run development server**
```bash
npm run dev
# or
yarn dev
```

4. **Build for production**
```bash
npm run build
# or
yarn build
```
